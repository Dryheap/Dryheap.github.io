Analyzing Success in the NBA

About this project

In this project, you can analyze success in the NBA both at the team and individual level. This dataset covers the NBA's 2021-2022 season's advanced statistics. The visualization system below will allow you to easily compare teams and players against each other and analyze which statisical categories in which they succeed. From this, you can determine which players and stats contribute the most when it comes to winning in the NBA.

If you need more information for any team, player, or statistic, there are multiple on-hover features implemented to help give you more context.

Team Analysis

Select a team from the Bar Chart to display that team on the associated Radar Chart. You can make multiple selections to compare their stats.
Selecting a team will also filter the Scatter Plot below to display only players for that selected team.

Player Analysis

Select a player from the Scatter Plot to display that player on the associated Radar Chart. You can make multiple selections to compare thair stats.
